# Log Analysis & Intrusion Detection

A lightweight, web-based application designed to analyze system logs and detect potential security intrusions or anomalous behavior. Built with Python, this tool parses log files to identify suspicious patterns, offering a clean interface for monitoring system security events.

## 🚀 Features

* **Automated Log Parsing:** Seamlessly reads and processes standard log files (e.g., access logs, system logs).
* **Threat Detection:** Analyzes log entries to identify potential intrusions, such as repeated failed login attempts, unauthorized access, or anomalous traffic patterns.
* **Web Dashboard:** Provides a user-friendly frontend to visualize analysis results, making it easier to review flagged events.
* **Customizable Rules:** Easy to extend Python logic to include custom security rules and thresholds for detection.

## 🛠️ Technologies Used

* **Backend:** Python, Flask (Web Framework)
* **Frontend:** HTML/CSS (Jinja2 Templating)
* **Data Input:** `.log` file processing
